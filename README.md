# Notes:

- Start localhost server for writing:
```
cd /root_path/to/project && middleman server
```

- Create a new blog post:
```
middleman article "The blog post title"
```

- Deploy to Github Pages:
```
middleman deploy
```

- Push to repo:
```
git push repo master
```

Do not push to origin!
